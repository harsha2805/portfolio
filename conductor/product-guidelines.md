# UX Principles
- **Balanced Motion:** High-intensity, immersive motion and interactivity for key moments, paired with subtle, functional animations that enhance focus elsewhere.
- **Inclusive Design:** Adhere to "reduced-motion" accessibility standards, providing a seamless experience for all users regardless of their preferences.
- **Immersive Experiences:** Utilize interactive components (Three.js, Globe, Grainient) to create a premium, tactile digital space.

# Visual Style
- **Sleek Minimalism:** A clean, high-contrast base with plenty of white space to emphasize the core content.
- **Tactile 3D Elements:** Depth-filled textures and materials within interactive sections to create a physical-like UI experience.
- **HoloStack Tokens:** Use the established HoloStack design tokens (glows, glassmorphism) to maintain a cohesive visual identity.

# Tone and Voice
- **Friendly & Engaging:** An approachable, warm, and inviting way of communicating with visitors, making technical achievements feel personal and accessible.
- **Clear & Expressive:** Ensure information is conveyed with clarity while remaining engaging and professional.

# Development Standards
- **Perf-First Development:** Mandatory performance optimization for all 3D, canvas, and heavy animation elements, using intersection observers and efficient asset loading.
- **Test-Driven Quality:** Comprehensive test coverage for all backend Laravel logic and critical frontend React flows.
- **Accessible-by-Design:** Strict adherence to semantic HTML and WCAG accessibility standards to ensure the site is usable for everyone.
- **Responsive & Adaptive:** Every component must be fully responsive and performant across all device types and screen sizes.
