# Programming Languages
- **PHP 8.2+:** Robust backend language powering the Laravel application.
- **TypeScript:** Typed JavaScript for predictable and scalable frontend development.

# Backend Framework
- **Laravel 12.0:** High-performance, expressive framework for backend routing, models, and business logic.
- **Pest PHP:** Modern testing framework for comprehensive backend test coverage.

# Frontend Framework
- **React 19.2:** Component-based library for building dynamic, interactive user interfaces.
- **React Router Dom 7.1:** Client-side routing for seamless navigation within the SPA.
- **Vite:** Next-generation frontend tooling for fast builds and hot-reloading.

# Styling and Interactivity
- **Tailwind CSS 4.0:** Utility-first CSS framework for rapid and consistent UI development.
- **Motion:** Comprehensive animation library for building immersive and fluid UI interactions.
- **GSAP:** Advanced animation tool for high-performance timeline-based motion.
- **Three.js:** 3D library for creating complex interactive graphics and visual effects.

# Database
- **SQLite:** Lightweight, serverless database used for development and local storage.

# Development Tools
- **Composer:** PHP dependency management.
- **npm:** JavaScript package management.
- **Artisan:** Laravel's built-in command-line interface.
